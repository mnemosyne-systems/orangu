# Debugging

Investigate the reported issue carefully.

Focus on:
1. Reproducing the problem from the available context
2. Identifying the most likely root cause
3. Calling out missing evidence or assumptions
4. Proposing the smallest defensible fix
5. Suggesting validation steps after the fix
